\section{Introduction}
Ce document est le \textbf{Quick Start Guide}. Il contient les étapes essentielles pour assembler, mettre sous tension et démarrer une mission d'inspection autonome avec l'Agri-Scout.

\section{Prérequis Matériels}
Pour démarrer le robot, il faut s'assurer d'avoir les éléments suivants :
\begin{itemize}
    \item 2 Batteries LiPo (pour les moteurs et ponts en H L298N).
    \item Powerbank USB-C (pour alimenter le Raspberry Pi de manière isolée afin d'éviter les chutes de tension "brown-outs").
    % \item Câble USB de données entre le Raspberry Pi et l'Arduino (le fil rouge 5V a été coupé pour isoler les alimentations, mais le GND commun est impératif).
\end{itemize}

\subsection{Démarrage Logiciel}

Pour se connecter au Raspberry Pi pour la première fois, vous devez utiliser un cable Ethernet pour connecter le Raspberry Pi à votre ordinateur.
D'abord, il faut metre en place des configurations sur votre ordinateur. Les instructions suivantes sont pour les systèmes Linux. Comme en general,
la langua de configuration est en anglais, les instructions sont écrites en anglais pour éviter les confusions.
\begin{enumerate}
    \item Open \textbf{Settings → Network};
    \item Locate the \textbf{Wired} connection;
    \item Open the IPv4 tab;
    \item Change \textbf{Method} from \textbf{Automatic (DHCP)} to \textbf{Shared to other computers};
    \item Click \textbf{Apply};
    \item Disable and re-enable the wired connection to restart the interface.
\end{enumerate}
Avec cette configuration, votre ordinateur agira comme un serveur DHCP pour le Raspberry Pi, lui attribuant une adresse IP automatiquement.

La deuxième étape est de se connecter au Raspberry Pi en utilisant SSH. Ouvrez un terminal et tapez la commande suivante :
\begin{verbatim}
arp -a
\end{verbatim}
Cette commande affichera une liste des appareils connectés à votre réseau local. 
Trouvez l'adresse IP associée au Raspberry Pi, généralement \textit{10.42.0.x} et connectez-vous en utilisant la commande SSH suivante :
\begin{verbatim}
ssh ubuntu@10.42.0.x
\end{verbatim}
Le mot de passe est : \textbf{agribot}.

Si l'utilisateur préfère une connexion sans fil, il est possible de changer la configuration 
du Raspberry Pi pour se connecter à un réseau Wi-Fi local, par exemple, d'un portable. 
Cependant, cela nécessite une configuration préalable du Raspberry Pi pour se connecter au réseau Wi-Fi.


Une fois connecté en SSH au Raspberry Pi, exécutez le script d'initialisation principal :
\begin{verbatim}
source /opt/ros/jazzy/setup.bash

cd ~/AGRI-Scout_Modif

source install/setup.bash

ros2 launch ~/AGRI-Scout_Modif/agri_launch.py
\end{verbatim}

Ce fichier se charge de lancer :
\begin{itemize}
    \item Le nœud de la caméra (V4L2).
    \item Le nœud du RPLiDAR.
    \item Le contrôleur de vision et de mouvement Python (\texttt{agri\_line\_follower.py}).
\end{itemize}
